# Manual Test Guide

本指南用于手动测试不同角色在不同环境下的功能行为。  
测试结果可用 ✅ (通过) / ❌ (未通过) / ⚠️ (异常)，并可填写备注。

### 📌 使用说明
1. **测试结果**  
   - ⬜ 未测试  
   - ✅ 通过  
   - ❌ 未通过  
   - ⚠️ 异常/需关注  

2. **备注**  
   - 可记录测试细节、错误信息或截图链接  
3. **扩展**  
   - 若新增功能，可直接在对应角色下添加新行或新表格
